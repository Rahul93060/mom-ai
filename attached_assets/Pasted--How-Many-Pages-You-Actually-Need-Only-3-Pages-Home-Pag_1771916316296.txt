📝 Step 3 – Create Form
forms.py

Example (modify based on your features):

from django import forms

class HealthForm(forms.Form):
    age = forms.FloatField()
    systolic_bp = forms.FloatField()
    diastolic_bp = forms.FloatField()
    bs = forms.FloatField()
    body_temp = forms.FloatField()
    heart_rate = forms.FloatField()

These fields MUST match training features.

🧠 Step 4 – Connect ML Model

Now the important part.

views.py
import pickle
import numpy as np
from django.shortcuts import render
from .forms import HealthForm

# Load model and encoder
model = pickle.load(open("mom_ai_model.pkl", "rb"))
encoder = pickle.load(open("label_encoder.pkl", "rb"))

def home(request):
    return render(request, "home.html")

def predict(request):
    if request.method == "POST":
        form = HealthForm(request.POST)
        if form.is_valid():

            data = [
                form.cleaned_data['age'],
                form.cleaned_data['systolic_bp'],
                form.cleaned_data['diastolic_bp'],
                form.cleaned_data['bs'],
                form.cleaned_data['body_temp'],
                form.cleaned_data['heart_rate'],
            ]

            prediction = model.predict([data])
            decoded_prediction = encoder.inverse_transform(prediction)

            return render(request, "result.html", {
                "risk": decoded_prediction[0]
            })
    else:
        form = HealthForm()

    return render(request, "form.html", {"form": form})

This is where ML connects to Django.

🚀 How It Works Flow

User opens home page

Clicks Start Prediction

Fills medical details

Submits form

Data goes to ML model

Model predicts risk

Encoder converts number to label

Result page shows risk

That’s the full pipeline.
